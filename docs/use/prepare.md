# 搭建环境

工欲善其事必先利其器,首先先选择您的开发工具,作者用的是`idea`,下面就以`ida`为例搭建开发环境

1. 下载安装[idea][download-idea],选择系统版本,下面作者以 `mac` 为例,`window`也同样适用;
![下载页面][download-intro]

2. 开始下载,拼网速的时候到了,耐心等待...
![下载完成][download-complete]

3. 开始安装 `idea` ,`mac` 直接拖动安装, `window` 下一步...,最后启动 `idea` 软件;
![开始安装][idea-install]
